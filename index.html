Missions réalisées

Mission 1 : Configurer et mettre en service un serveur NAS pour de la sauvegarde

Contexte : Dans le cadre de mon stage au sein du Groupe Jeulin Immobilier, j’ai été chargé d'optimiser la stratégie de sauvegarde des données critiques de l'entreprise. L’organisation regroupe plusieurs pôles (Comptabilité, Communication, Commercial, Immobilier, Évènementiel) dont les fichiers sont stockés sur des serveurs de production. L'objectif était de rajouter du stockage sur le serveur NAS de Backup pour pouvoir accueillir plus de données.

Technologies : NAS Synology, Active Backup for Business (Synology), Microsoft OneDrive, RAID 5, QNAP Qfinder, Seagate IronWolf, Authentification Multi-Facteurs (AMF)

Productions :

  • Guide de paramétrage

Compétences mobilisées : B1.1 B1.2 B1.5

Mission 2 : Création de profil sous Active Directory

Contexte : Dans le cadre de mon stage au sein du Groupe Jeulin Immobilier, j’ai été chargé de la création d'un profil utilisateur sur l'Active Directory au sein de l'entité Ouest Bureau

Technologies : Active Directory, gestion des droits d'accès

Productions :

  • Document d'explication de la création du profil

Compétences mobilisées : B1.1

Mission 3 : Infrastructure Groupe Jeulin Immobilier

Contexte : Dans le cadre de mon stage au sein du Groupe Jeulin Immobilier, j'ai eu pour objectif de concevoir un schéma de l'infrastructure réseau de l'entreprise, afin de mieux appréhender le fonctionnement des technologies et des équipements en place.

Technologies : Microsoft Visio

Productions :

  • Création d'un schéma réseau de l'entreprise

Compétences mobilisées : B1.1

Mission 4 : Gestion de parc informatique

Contexte : Dans le cadre de mon stage, j'ai eu pour mission de participer à la gestion du parc informatique de l'entreprise. Cela m'a amené à prendre en main l'outil de supervision NinjaOne, à améliorer l'inventaire des serveurs et postes clients, à préparer des postes de travail et ordinateurs portables professionnels, ainsi qu'à effectuer un comparatif de bornes Wi-Fi disponibles sur le marché.

Technologies : NinjaOne outil de gestion de parc informatique à distance RMM (Remote Monitoring and Management)

Productions :

  • Ninja One Outil de gestion de parc informatique RMM et de supervision
  • Inventaire parc serveur/postes clients
  • Préparation de postes et d'ordinateur portable professionnel
  • Comparatif bornes WIFI

Compétences mobilisées : B1.1 B1.2 B1.5

Missions réalisées

Mission 1 : Mise en place d'une infrastructure réseau de test local isolé

Contexte : Dans le cadre de mon stage, j'ai eu accès à un ensemble d'équipements réseau afin de monter un réseau local isolé du réseau de production de l'entreprise. Ce laboratoire me permet de travailler les compétences techniques suivantes :

  • Modélisation réseau – Simuler, schématiser et comprendre un environnement réseau local
  • Câblage & infrastructure – Installation et raccordement des liaisons réseau situées sur différents niveaux du bâtiment (switch, serveur, prise murale, rocade)
  • Virtualisation – Mise en place de machines virtuelles sous Hyper-V, création de commutateurs virtuels externes et internes
  • Pare-feu & NAT – Paramétrage de règles de filtrage et configuration du NAT
  • Tests de connectivité – Vérification de la connectivité et validation du blocage des flux
  • Switch & VLANs – Configuration d'un switch manageable et segmentation par VLANs
  • Wi-Fi – Configuration d'une borne d'accès sans fil

Technologies : NinjaOne, Windows Server 2022, Active Directory, DNS, Hyper-V, PowerShell, Machines virtuelles (VM), Switch Aruba 1930, VLANs, Mode Trunk/Access, DHCP, IP statique, Routage inter-VLAN, Routeur/pare-feu Sophos XG85 virtuel, Règles de filtrage, NAT, VSWITCH, Borne Wi-Fi Sophos APX320

Productions :

  • Schéma réseau
  • Configuration de VSwitch sur Hyper-V
  • Configuration de routeur/pare-feu Sophos
  • Configuration commutateur Aruba
  • Configuration borne Wi-Fi Sophos
  • Installation du rôle Active Directory sur un serveur virtuel

Compétences mobilisées : B1.1 B1.4 B1.5 B1.6

Technologies : Serveur HP ProLiant

Productions :

  • Installation du système d'exploitation Windows Server 2022
  • Installation du rôle Hyper-V sur Windows Server 2022
